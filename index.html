Overview

General Flow

Service Account

Before you can run the Python scripts you'll need to set up a Service Account in GAFE. This is a robot account that your script will use to access Sheets. Any Sheet that you want your script to access will be shared with this Service Account.

MySQL

You must also set up a MySQL database on the same server that the Python script will be running. I've included a script to that should take care of most of this part.
